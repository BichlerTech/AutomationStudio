# Automation Studio

## Requirements
```
- Eclipse 2019-06 (https://www.eclipse.org/downloads/packages/release/2019-06/r)
  or
- Eclipse 2020-03 (https://www.eclipse.org/downloads/packages/release/2020-03)
- Java 1.8.0_201 (tested version)
```
