addFolderNode((int), (String), (int), (String), (String), (String), (String), (byte), (int), (int))

Creates a new Object as Folder and attaches it to the parent node.
Parameters:
Name Type Description
parentNsIndex (int) Namespace index of parent node.
parentNodeid (String) Nodeid of parent in form ("i=1" or "s=demo")
newNsIndex (int) Namespace index of requested new node.
newIndex (String) Nodeid of requested new node. e.g. "i=1" or "s=demo"
browseName (String) Name to browse this folder node. e.g. "demo.myfolders.folderinterval"
description (String) Description of this folder.
displayName (String) Name which will be display in our OPC UA Browser.
eventNotifier (byte) Notifier type which should raise an event for the folder.
writeMask (int) Mask to write from system to this node.
userWriteMask (int) Mask to write as a user connection to this node.

Example

/**
* test call of addFolderNode
*/
addFolderNode(0, "i=87", 1, "i=1", "simulation", "new Folder for test variables", "simulation", 1, 2, 3);
